## TP1 — Gestión de la configuración y programación Python

### Descripción

Este trabajo práctico aborda la instalación y uso de herramientas de gestión de configuración de software, junto con el desarrollo de programas en Python.

### Herramientas utilizadas

- Git
- Python 3
- Pip3
- Matplotlib

### Programas desarrollados

1. `primes.py` — Cálculo e impresión de números primos
2. `factorial.py` — Cálculo de factorial con soporte de rangos
3. `factorial_OOP.py` — Versión orientada a objetos del cálculo de factorial
4. `collatz.py` — Cálculo y graficación de la conjetura de Collatz

---

## Instrucciones de uso

### Requisitos previos

- Python 3 instalado ([descargar desde python.org](https://www.python.org/downloads/))
- Git instalado
- Matplotlib instalado (`pip install matplotlib`)

### Cómo ejecutar los programas

#### primos.py

```bash
python3 src/primos.py
```

#### factorial.py

```bash
# Con un número específico
python3 src/factorial/factorial.py 10

# Con un rango
python3 src/factorial/factorial.py 4-8

# Sin argumentos (solicita ingreso manual)
python3 src/factorial/factorial.py
```

#### collatz.py

```bash
python3 src/collatz/collatz.py
```
